✦ Features

Everything your team needs,
nothing they don't.

Built lean and fast — no bloat, no steep learning curve. Just dashboards, organised and accessible.

📤

Upload Any Dashboard

Upload HTML reports or paste Power BI, Excel Online, Looker, Tableau, or any embed URL — it just works.

🔐

Role-Based Access

Create custom roles with PIN access. Each team only sees the departments assigned to them — nothing more.

🌙

Light & Dark Themes

Professional light theme by default. One click switches to a rich dark mode — preference saved per user.

⚡

Instant Search

Real-time search across all dashboards — scoped to the user's role. Find anything in milliseconds.

⭐

Favourites & Recent

Each user can star frequently used dashboards. Recently viewed appears automatically in Quick Links.

📋

Activity Feed

Live audit trail of recent uploads, edits, and updates. Always know what changed and when.

✦ How It Works

Up and running in minutes.

From first login to a fully organised dashboard hub — four simple steps.

1

Admin Logs In

Siby enters the Admin PIN. Full edit access is granted instantly.

2

Add Dashboards

Upload HTML files or paste embed URLs per department. Done in seconds.

3

Create Team Roles

Define roles like "HR Team" or "Finance Viewers" with custom PINs and department access.

4

Teams Access

Staff open the URL, pick their role, enter PIN — and see only their dashboards.
