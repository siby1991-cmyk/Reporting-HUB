✦ How It Works

Up and running in minutes.

From first login to a fully organised multi-department hub — four simple steps.

1

Admin Logs In

Enter Admin PIN. Full edit access is granted instantly across all departments.

2

Add Departments & Dashboards

Create departments, upload HTML files or paste embed URLs. Done in seconds.

3

Create Team Roles

Define roles like "HR Team" or "Finance Viewers" with custom PINs and scoped access.

4

Teams Access Their Hub

Staff open the URL, pick their role, enter PIN — and see exactly their dashboards.
